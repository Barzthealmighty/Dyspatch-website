# Dyspatch — Website

> **"Delivering trust, one ride at a time."**

The official website for [Dyspatch](https://dyspatch.com.ng) — a technology-first, asset-light last-mile logistics platform connecting senders with verified dispatch riders across Lagos and Benin, Nigeria.

## 🚀 Live Site

Visit: [dyspatch.com.ng](https://dyspatch.com.ng)

## 📦 Project Structure

```
dyspatch-website/
├── index.html          # Complete SPA (all 11 pages)
├── app-onboarding.jpg  # App onboarding screenshot
├── app-splash.jpg      # App splash screen
├── README.md           # Documentation
└── .gitignore
```

## 🛠 Tech Stack

- **Pure HTML/CSS/JS** — single-file SPA, no build tools
- **Montserrat** via Google Fonts
- **Leaflet.js** + CARTO dark tiles for real Lagos mapping
- **Hash-based routing** for page navigation
- **Glassmorphism** UI with backdrop-filter
- **Fully responsive** — mobile, tablet, desktop

## 📄 Pages

Home · Track · Become a Rider · How It Works · For Businesses · About · Contact · Help Centre · Blog · Privacy Policy · Terms of Service
